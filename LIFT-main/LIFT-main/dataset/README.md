ECL, Weather, Traffic, ETT, Illness benchmarks can be downloaded from
[Google Drive](https://drive.google.com/drive/folders/1ZOYpTUa82_jCcxIdTmyr0LXQfvaM9vIy) provided by Autoformer.

The Solar benchmark can be downloaded from 
[Github](https://github.com/laiguokun/multivariate-time-series-data) provided by MTGNN.

The PeMSD8 benchmark can be downloaded from 
[Github](https://github.com/guoshnBJTU/ASTGNN/tree/main/data/PEMS08) provided by ASTGCN.

The Wind benchmark can be downloaded from
[Kaggle](https://www.kaggle.com/datasets/sohier/30-years-of-european-wind-generation). We only use the last 50000 rows.
